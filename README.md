## 📌 Project Overview
This project is a Machine Learning-based Car Price Prediction system built using Python. The model analyzes various features such as manufacturing year, fuel type, transmission type, and kilometers driven to predict the price of a car.

It demonstrates a complete end-to-end regression workflow, including data preprocessing, visualization, model training, and evaluation.

---

## 🎯 Objective
- Predict car prices using machine learning techniques
- Understand regression algorithms like Linear Regression and Lasso
- Gain hands-on experience with real-world datasets

---

## 🛠️ Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow
1. **Data Collection**
   - Used a dataset containing car details and pricing information

2. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical variables
   - Feature selection

3. **Data Visualization**
   - Used **Matplotlib** and **Seaborn** to analyze patterns and relationships

4. **Train-Test Split**
   - Used `train_test_split` to divide the dataset into training and testing sets

5. **Model Building**
   - Implemented **Linear Regression** and **Lasso Regression** models

6. **Model Evaluation**
   - Evaluated performance using appropriate regression metrics

---

## 📊 Features
- Predicts car prices based on input features
- Uses both basic and regularized regression techniques
- Includes data visualization for insights
- Clean and structured ML pipeline
